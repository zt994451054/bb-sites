This repository stores only private Xiaohongshu adapter overrides for `bb-browser`.

Clone it into `~/.bb-browser/sites` on the target machine:

```bash
git clone https://github.com/zt994451054/bb-sites.git ~/.bb-browser/sites
```

Only `xiaohongshu/*` commands are overridden. Other sites continue using the community adapters from `~/.bb-browser/bb-sites`.
